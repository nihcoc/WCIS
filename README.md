**WCIS – DIY Irrigation System Using ESP32**
---
WCIS (Wirelessly Controlled Irrigation System) is an ESP32-based smart irrigation controller designed to control a solenoid valve or a SERVO for turning water flow ON and OFF on a daily schedule or manually.

WCIS uses an ESP32 microcontroller to control a 25KG servo through a MOSFET driver circuit and GPIO.
MOSFET is used to control power OUT to the servo as when idle the servo might use power.

Housing is modelled for my valve and should be adjusted according to the valve you have before printing.
