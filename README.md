**WCIS – DIY Irrigation System Using ESP32**
---
WCIS (Wirelessly Controlled Irrigation System) is an ESP32-based smart irrigation controller designed to control a solenoid valve or a SERVO for turning water flow ON and OFF automatically or manually.

WCIS uses an ESP32 microcontroller to control a DC solenoid valve or a servo through a MOSFET driver circuit.

The housing can be printed with PLA if you plan not to expose it to the elements.
